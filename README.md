# Price Aggregator

# Using

$ `git clone https://github.com/vmalits/jobs-monitoring.git`

$ `cd jobs-monitoring`

$ `cp .env.example .env`

$ `sail up -d`

$ `sail php artisan key:generate`

$ `sail composer install`

$ `sail php artisan migrate`

$ `sail php artisan test`

# Info
project running on: `http://localhost`
